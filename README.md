# FPGA OV7670 Edge Detection

Real-time image capture and edge detection pipeline implemented in FPGA using the OV7670 camera module and Xilinx development board.

---

## Project Overview

This project captures live image data from the OV7670 camera module,
processes the image stream inside the FPGA and applies an edge detection filter
in real time.

The goal of the project was to explore FPGA-based image processing,
hardware-oriented design and real-time digital systems.

---

## Main Features

- Real-time image capture from OV7670 camera
- FPGA-based edge detection filter
- RTL hardware design
- Image processing pipeline
- Hardware-software integration
- Simulation and verification

---

## Technologies Used

- FPGA

## System Architecture

Camera Module (OV7670)

→ FPGA Output Pipeline
